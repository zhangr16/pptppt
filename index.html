Vue.js简介

 

  • Vue.js是一个构建数据驱动的 web 界面的框架
  • Vue.js 的目标是实现响应的数据绑定和组合的视图组件
  • Vue.js 的核心是一个响应的数据绑定系统

Vue实例

var vm = new Vue({
    el: '#demo',
    data: {
        a: 1,
        b: 2
    },
    created: function () {

        // `this` 指向 vm 实例
        console.log('a is: ' + this.a)
    }
})

包含数据、模板、挂载元素、方法、生命周期钩子等选项
